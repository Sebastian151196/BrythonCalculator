Calculadora usando Brython

Brython está diseñado para reemplazar Javascript como el lenguaje de script para la Web. Como tal, es una implementación de Python 3 (puede tomarlo para una prueba de manejo a través de una consola web), adaptada al entorno HTML5, es decir, con una interfaz para los objetos y eventos DOM.

Calculadora

Realice una página Web, usando Bootstrap, JavaScript y Brython, que simule una calculadora básica. La calculadora debe simular las operaciones de una calculadora como igual, suma, resta, división, multiplicación, porcentaje, elevar un numero al cuadrado y calcular la raíz cuadrada de un número. Debe incluir un botón para los número (del 0 al 9), botón de punto decimal, botón para cambiar de signo, botón para borrar una cifra (CE) y botón para inicializar la calculadora(C).
